## 🎯 Project Objective

The goal of this project is to develop a web-based Hotel Room Booking System that allows users to easily browse available rooms and book them online. The system will also provide an admin panel to manage rooms and bookings efficiently.

---

## 🧩 System Overview

The system will consist of two main parts:

* **User Panel** for customers
* **Admin Panel** for administrators

It aims to replace manual booking systems with a faster, more organized digital solution.

---

## 🌐 Type of System

Online Booking and Management System

---

## 📦 Scope of the Project

### Included Features

* User registration and login
* View available hotel rooms
* Online room booking
* Booking history
* Admin management of rooms and bookings

### Not Included (Initial Version)
